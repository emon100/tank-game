# tank-game
## Introduction
A tank game for school project.

2019年7月东北大学c++课设,使用了Qt5编写(2020年之后版本不确定支持),最新版本可支持Qt6.0编译运行。

除了游戏外还具有一个地图编辑器。

tank_game.pro:qt项目
game文件夹中为游戏的源代码
interface文件夹中为界面的源代码
mapmaker文件夹中为地图编辑器的源代码
